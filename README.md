# PADÌ website (static)

Two-page static site with an earthy, sustainable vibe.

## Run locally

- Open `index.html` in your browser, or serve the folder:

```bash
python3 -m http.server 5173
```

Then visit `http://localhost:5173`.

## Customize

- Update contact + social links in `connect.html`.
- Replace logo placeholders in `index.html` (`#featured` section).
- Replace recent news cards in `index.html` (`#news` section).
- Swap illustrations in `assets/` (SVGs) as your branding evolves.
